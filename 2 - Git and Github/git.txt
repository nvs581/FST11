- Initialize a repository
git init 

- Configure the username in git
git config user.name "GitHub Username"

- Configure the email in git
git config user.email "GitHub Email"

- Check configured username in git
git config user.name

- Check configured email in git
git config user.email

- Check the status of change in files/folders
git status

- Add all untracked files to the staging area
git add .

- Cornmit L files from staging area
git commit -m "description of your change"

Legends
U - untracked
A - added
M - modified
D - deleted
R - renamed

- Show commit history
git log
- Show comit history (summarized)
git log --onelinegi

- Clear terminal commands
clear

- create a branch
git branch <branchName>

- list all branched
git branch -a

- change branch
git checkout <branchName>
git switch <branchName> - newer but not works with all older versions of git

- Clear terminal commands
clear

- Merge feature branch to main/master branch
git merge <branchName>

delete branch
- git branch -d <branchName>

- Add a remote to local repository
git remote add origin "repository-url"
- Set branch to main
git branch -M main

- Push local repository to remote repository (first time push)
git push -u origin main

- confirm connection to remote repository
git remove -v

- push local repository to remote repository (once connection has been established)
git push
